# Changelog
All notable changes to HELPDESK.AI will be documented in this file.

## [Unreleased]
### Added
- AI-powered ticket classification and auto-routing
- WebSocket support for real-time ticket updates
- Mobile app (React Native) for on-the-go support
- Dark mode support across all dashboard components
- Supabase integration for backend data persistence
### Changed
- Migrated frontend to Next.js with App Router
- Updated backend to FastAPI with async support
### Fixed
- Duplicate detection now learns from saved tickets in Supabase
- Dashboard components now respect dark mode settings

## [1.0.0] - 2025-01-15
### Added
- Initial release of HELPDESK.AI
- Basic ticket management system (create, view, update, delete)
- User authentication and role-based access control
- Email notifications for ticket updates
- Basic dashboard with ticket statistics
- REST API for ticket CRUD operations
